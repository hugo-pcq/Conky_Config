# Conky_Config
Optimised for Lenovo X1 Carbon, Ubuntu 22.04 LTS

Download conkyrc file in your /home folder, and hide it (mv conkyrc .conkyrc)

sudo apt-get install conky-all

Add /usr/bin/conky to Startup Application (GUI)

reboot
